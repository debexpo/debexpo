You need to use dput to upload packages. Add an entry like the following to your ~/.dput.cf:

[debexpo]
fqdn = expo.debian.net
incoming = /upload/email@address/yourpassword
method = http
allow_unsigned_uploads = 0

You should change the email@address and yourpassword entries with the email address and password you use to login.

Now you should execute:

dput debexpo package_version_source.changes
